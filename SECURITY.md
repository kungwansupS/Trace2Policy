# Security Policy

Trace2Policy v0.1 is alpha software. Security fixes are accepted for supported
versions listed below.

## Reporting a Vulnerability

Please do not disclose sensitive vulnerabilities in public issues. Until a
private reporting channel is published, contact the repository owner directly
through GitHub.

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | Yes       |
| < 0.1   | No        |
