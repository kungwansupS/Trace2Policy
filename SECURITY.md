# Security Policy

TracePolicyKit v0.2 is alpha software. Security fixes are accepted for supported
versions listed below.

## Reporting a Vulnerability

Please do not disclose sensitive vulnerabilities in public issues. Until a
private reporting channel is published, contact the repository owner directly
through GitHub.

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.2.x   | Yes       |
| < 0.2   | No        |
